# 墨唐家庭记账（FamilyBill）UI设计方案

## 1. 设计原则

### 1.1 核心设计理念
- **轻量化**：保持界面简洁，避免过度设计，减少视觉干扰
- **易用性**：操作流程简单直观，降低学习成本
- **一致性**：Web端和App端设计风格统一，提供连贯的用户体验
- **响应式**：适配不同屏幕尺寸，确保在各种设备上都能良好展示
- **高效性**：核心功能一键可达，减少操作步骤

### 1.2 设计风格
- **主色调**：#409EFF（蓝色）- 代表信任、专业和稳定
- **辅助色**：#67C23A（绿色）- 代表收入、增长和积极
- **警示色**：#F56C6C（红色）- 代表支出、警告和重要
- **中性色**：#303133（深灰）、#606266（中灰）、#909399（浅灰）- 用于文本和边框
- **背景色**：#F5F7FA（浅灰蓝）- 提供舒适的视觉体验

### 1.3 排版规范
- **字体**：系统默认无衬线字体（如Roboto、San Francisco、PingFang SC）
- **标题层级**：
  - H1：20px，加粗
  - H2：18px，加粗
  - H3：16px，加粗
  - 正文：14px
  - 辅助文字：12px
- **行高**：1.5-1.8倍

## 2. Web端设计

### 2.1 整体布局
- **布局结构**：左侧导航栏 + 顶部状态栏 + 主内容区
- **响应式设计**：
  - 大屏幕（≥1200px）：三栏布局
  - 中屏幕（992px-1199px）：三栏布局，导航栏宽度减小
  - 小屏幕（768px-991px）：折叠左侧导航栏，通过顶部菜单按钮展开
  - 超小屏幕（<768px）：单栏布局，导航栏完全折叠

### 2.2 页面设计

#### 2.2.1 登录页面
- **布局**：居中卡片式设计
- **元素**：
  - 项目Logo
  - 登录标题
  - 用户名/邮箱输入框
  - 密码输入框
  - 记住密码选项
  - 忘记密码链接
  - 登录按钮
  - 注册链接

#### 2.2.2 仪表盘
- **布局**：卡片式网格布局
- **元素**：
  - 顶部统计卡片（总收入、总支出、结余）
  - 收支趋势图（折线图）
  - 最近收支记录列表
  - 分类占比图（饼图）
  - 快捷记账入口

#### 2.2.3 收支记录
- **布局**：表格+筛选栏
- **元素**：
  - 顶部筛选栏（日期范围、收支类型、分类、账户）
  - 新增记录按钮
  - 收支记录表格（日期、类型、分类、金额、账户、备注）
  - 分页控件

#### 2.2.4 新增/编辑收支记录
- **布局**：表单式设计
- **元素**：
  - 收支类型选择（支出/收入/转账）
  - 金额输入框
  - 分类选择器
  - 账户选择器
  - 日期选择器
  - 备注输入框
  - 标签选择器
  - 保存/取消按钮

#### 2.2.5 账户管理
- **布局**：卡片式列表
- **元素**：
  - 账户列表（名称、类型、余额、操作）
  - 新增账户按钮
  - 账户编辑/删除功能

#### 2.2.6 分类管理
- **布局**：树状结构+表单
- **元素**：
  - 分类树（一级分类+二级分类）
  - 新增分类按钮
  - 分类编辑/删除功能
  - 分类图标选择

#### 2.2.7 统计分析
- **布局**：多图表组合
- **元素**：
  - 统计时间范围选择
  - 收支对比图（柱状图）
  - 分类占比图（饼图）
  - 账户收支图（堆叠柱状图）
  - 趋势分析图（折线图）
  - 数据导出按钮

#### 2.2.8 成员管理
- **布局**：表格+表单
- **元素**：
  - 成员列表（姓名、角色、权限、操作）
  - 邀请成员按钮
  - 成员编辑/删除功能
  - 权限设置

#### 2.2.9 系统设置
- **布局**：标签页式设计
- **元素**：
  - 基本设置（语言、货币、日期格式）
  - 安全设置（密码修改、登录设备）
  - 备份恢复
  - 通知设置

## 3. App端设计

### 3.1 整体布局
- **布局结构**：底部导航栏 + 主内容区
- **导航项**：首页、记账、统计、我的

### 3.2 页面设计

#### 3.2.1 登录/注册页面
- **布局**：全屏垂直布局
- **元素**：
  - 项目Logo
  - 登录/注册表单
  - 第三方登录选项
  - 忘记密码链接

#### 3.2.2 首页（仪表盘）
- **布局**：卡片式滚动布局
- **元素**：
  - 顶部统计卡片（总收入、总支出、结余）
  - 快捷记账按钮（支出/收入）
  - 收支趋势图（折线图）
  - 最近收支记录
  - 分类占比快捷查看

#### 3.2.3 记账页面
- **布局**：全屏表单式设计
- **元素**：
  - 金额输入框（大字号）
  - 收支类型切换（支出/收入）
  - 分类选择（图标+文字）
  - 账户选择
  - 日期选择
  - 备注输入
  - 标签选择
  - 保存按钮

#### 3.2.4 统计页面
- **布局**：标签页+图表组合
- **元素**：
  - 统计时间范围选择
  - 标签页切换（概览、分类、账户、趋势）
  - 各类统计图表
  - 数据导出按钮

#### 3.2.5 我的页面
- **布局**：列表式设计
- **元素**：
  - 用户信息卡片（头像、昵称、ID）
  - 功能列表（账户管理、分类管理、成员管理、预算管理、数据备份、设置）
  - 关于我们、帮助中心、退出登录

## 4. 组件设计

### 4.1 通用组件
- **按钮**：主要按钮、次要按钮、文本按钮、图标按钮
- **输入框**：文本输入、数字输入、密码输入、日期选择器
- **选择器**：下拉选择器、级联选择器、标签选择器
- **卡片**：统计卡片、内容卡片、列表卡片
- **表格**：基础表格、可编辑表格、分页控件
- **图表**：折线图、柱状图、饼图、雷达图
- **弹窗**：模态框、抽屉式弹窗、Toast提示

### 4.2 业务组件
- **记账表单**：快速记账组件、完整记账表单
- **统计卡片**：收支统计、余额统计
- **分类选择器**：带图标和颜色的分类选择
- **账户卡片**：显示账户余额和类型
- **收支记录项**：列表形式的收支记录展示

## 5. 交互设计

### 5.1 核心交互流程
- **记账流程**：点击记账按钮 → 选择收支类型 → 输入金额 → 选择分类和账户 → 保存记录
- **统计查看流程**：进入统计页面 → 选择时间范围 → 切换统计维度 → 查看图表详情
- **账户管理流程**：进入账户页面 → 点击新增账户 → 填写账户信息 → 保存

### 5.2 微交互设计
- **按钮悬停效果**：颜色加深、轻微放大
- **输入框焦点效果**：边框颜色变化、阴影效果
- **加载状态**：骨架屏、加载动画
- **成功/失败反馈**：Toast提示、状态变化
- **页面切换动画**：平滑过渡、淡入淡出

## 6. 响应式设计

### 6.1 Web端响应式适配
- **大屏（≥1200px）**：完整三栏布局，多列卡片展示
- **中屏（992px-1199px）**：三栏布局，导航栏宽度减小，卡片调整为2列
- **小屏（768px-991px）**：折叠左侧导航栏，主内容区自适应，卡片调整为1-2列
- **超小屏（<768px）**：单栏布局，导航栏完全折叠，卡片单列展示

### 6.2 App端适配
- **手机端**：适配不同屏幕尺寸，确保内容居中展示
- **平板端**：优化布局，充分利用屏幕空间，展示更多内容
