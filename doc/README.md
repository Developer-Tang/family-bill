# 墨唐家庭记账（FamilyBill）

## 项目概述
墨唐家庭记账（FamilyBill）是一款专注于家庭和个人财务管理的轻量化记账软件，致力于提供简单易用、功能丰富的记账体验，帮助用户轻松掌握财务状况，实现科学理财。

## 1. 项目目标
开发一套全功能家庭记账软件，支持多成员协作管理收支，通过 Go 后端、Vue 管理端、UniApp 移动端实现跨端同步，满足"快速记账、智能分析、灵活部署、安全可靠"的核心需求，覆盖个人用户、小家庭和小型团队等多种场景。

## 2. 核心价值
- **极致记账体验**：支持快捷模板、周期记账、日历记账等多种记账方式，30秒内完成收支录入，大幅提升记账效率
- **多维度数据分析**：提供收支结余表、类别报表、账户报表、成员报表、预算报表等7种报表类型，通过图表直观展示财务状况，辅助科学决策
- **灵活的协作模式**：支持家庭共享账本与个人私密账本，精细化权限控制，满足不同角色（记账人、管账人、查账人）的协作需求
- **数据安全保障**：采用数据加密存储、传输加密、访问权限控制、操作日志审计等多重安全机制，确保用户数据隐私安全
- **全面的功能覆盖**：从基础记账到高级功能（预算管理、账单导入、多币种支持、资产管家等），满足用户全方位的财务管理需求
- **轻量化灵活部署**：支持单机轻量部署（SQLite）与多设备协作（MySQL），Docker容器化部署，成本可控，适合个人和小型团队使用

## 3. 技术架构
- **后端**：Go 1.21+，框架Gin（轻量易扩展），ORM采用GORM（支持多数据库适配）
- **前端管理端**：Vue3 + Element Plus + TS，构建工具Yarn、Vite，图表库ECharts
- **移动端**：Uniapp（支持微信小程序/Android/iOS），UI组件库uView 3
- **数据库**：默认SQLite（单文件，零配置），支持切换至MySQL 8.0+（通过配置）
- **部署**：Docker容器化，前后端同容器部署，环境变量动态配置

## 4. 主要功能模块
- **用户与权限管理**：注册登录、家庭组管理、账本权限控制、成员角色细分、操作日志审计
- **账户管理**：支持多种账户类型、余额管理、转账功能、多币种支持
- **收支记录**：多种记账类型（支出、收入、转账、借贷等）、快速录入、日历记账、周期记账、流水锁定解锁、离线支持
- **统计分析**：数据看板、多维度图表展示、7种专业报表类型、多维度筛选
- **数据同步与备份**：多端实时同步、数据导出（Excel/CSV）、自动备份恢复、数据安全保障
- **高级功能**：预算管理、账单导入（标准/自定义）、标签体系、多账本管理、商家管理、项目管理、资产管家、回收箱、账单相册、财务云盘

## 5. 典型应用场景
- **家庭日常财务管理**：买菜后用手机快速录入支出；家庭成员共同管理家庭账本，自动同步记账数据
- **财务分析与规划**：月度复盘查看各类消费占比，通过多维度报表分析消费习惯；设置月度预算，超支时收到提醒
- **多成员费用分摊**：记录房租、水电费等共同支出，自动计算各成员分摊金额；支持按比例或固定金额灵活分摊
- **个人资产管理**：通过资产管家功能，全面了解个人资产负债状况；定期导出数据备份，用于个人财务规划
- **特殊场景应用**：多账本独立管理（日常/旅行/项目）；一键导入微信/支付宝账单；账单图片与记账记录关联存储
- **小型团队财务管理**：小团队共享账本，设置不同成员权限；记录项目开支，生成项目专用报表
